# Bandit Level 0 → Level 1

## 🎯 Objectif
Trouver le mot de passe du niveau suivant, stocké dans un fichier appelé `readme` situé dans le répertoire home.

## 🧠 Raisonnement
La navigation en ligne de commande repose sur trois réflexes fondamentaux : savoir où on est, voir ce qu'il y a autour de nous, et lire ce qui nous intéresse. En arrivant sur le serveur, j'ai d'abord vérifié ma position, puis listé le contenu du dossier pour repérer le fichier `readme`, et enfin lu son contenu pour y trouver le mot de passe.

## 🔧 Commandes utilisées

| Commande | Syntaxe | Description |
|----------|---------|-------------|
| `pwd` | `pwd` | Affiche le chemin du dossier courant (Print Working Directory) |
| `ls` | `ls` | Liste les fichiers et dossiers présents dans le répertoire courant |
| `cat` | `cat <fichier>` | Affiche le contenu d'un fichier dans le terminal |

## 💡 Difficultés rencontrées
Aucune difficulté particulière sur ce niveau. C'est une introduction aux commandes de base de navigation Linux.

## 📚 Ce que j'ai appris
La navigation Linux repose sur un enchaînement logique : on sait où on est avec `pwd`, on regarde ce qu'il y a avec `ls`, et on lit ce qui nous intéresse avec `cat`. C'est la base de tout déplacement et exploration dans un terminal.
