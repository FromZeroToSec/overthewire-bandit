# Bandit Level 00

## 🎯 Objectif
Se connecter au serveur via SSH sur un port spécifique.

## 🧠 Raisonnement
J'ai dû faire des recherches pour comprendre 
comment me connecter en SSH et comment préciser 
un port différent du port par défaut.

## 🔧 Commandes utilisées
| Commande | Syntaxe | Description |
|----------|---------|-------------|
| `ssh` | `ssh user@host -p port` | Se connecter à un serveur distant via SSH |

## 💡 Difficultés rencontrées
Je ne savais pas utiliser SSH ni comment 
spécifier un port de connexion.

## 📚 Ce que j'ai appris
SSH est un protocole de connexion à distance sécurisée.
Le port par défaut de SSH est le 22. 
L'option `-p` permet de préciser un port différent.
