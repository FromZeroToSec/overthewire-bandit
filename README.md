## 🔧 Commandes apprises

| Commande | Description | Maîtrisée |
|----------|-------------|-----------|
| `ssh user@host -p port` | Connexion distante sécurisée via SSH | ✅ |
| `pwd` | Affiche le répertoire courant | ✅ |
| `ls` | Liste les fichiers et dossiers | ✅ |
| `ls -a` | Liste tous les fichiers, y compris les cachés | ✅ |
| `ls -l` | Liste les fichiers avec les détails (taille, permissions...) | ✅ |
| `cat <fichier>` | Affiche le contenu d'un fichier | ✅ |
| `cat ./<fichier>` | Lit un fichier dont le nom commence par `-` | ✅ |
| `cd <dossier>` | Navigue dans un dossier | ✅ |
| `cd ..` | Remonte d'un niveau dans l'arborescence | ✅ |
| `file <fichier>` | Identifie le type d'un fichier (ASCII text, data...) | ✅ |
| `find -size <n>c` | Cherche des fichiers par taille exacte en bytes | ✅ |
| `find / -user <u> -group <g> -size <n>c` | Cherche un fichier sur tout le serveur selon plusieurs critères | ✅ |
| `2>/dev/null` | Redirige les erreurs (stderr) vers la poubelle | ✅ |
| `*` (wildcard) | Remplace n'importe quelle suite de caractères | ✅ |
| `mkdir <dossier>` | Crée un nouveau dossier | ✅ |
| `touch <fichier>` | Crée un fichier vide | ✅ |
| `grep <mot> <fichier>` | Recherche une chaîne de caractères dans un fichier | ✅ |
| `du <fichier>` | Affiche la taille d'un fichier ou dossier | ✅ |
